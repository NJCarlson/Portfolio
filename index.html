SIMETRI M3TS Care Under Fire Training

SIMETRI has developed a virtual environment surrounding a wounded soldier while engaged in active combat with a virtual enemy combatant. In this environment, the trainee must apply a tourniquet and return enemy fire when the situation calls for it. Augmented Reality (AR) plays a role by overlaying a pool of blood underneath the soldier’s wounded leg. The trainee’s weapon, medical instruments, and the wounded soldier provide haptic cues to the trainee while the virtual environment overlaid onto the physical scene and simulated physiology are provided by the M3TS system.

SIMETRI CPR Training

SIMETRI created a module to assist health care professionals in the training and practice of CPR. This unit consisted of a box with a censor, connected to an app via Wifi. The application provides training instructors to see the compression depth and rate in real time.

What I worked on:

I was the lead developer on this project. I created the main application in Unity using C#, and the embedded software using C++. Not only did I write all the code for this project and implement all UI assets, I also soldered, flashed, and assembled all units!

Data Thief - A VR Puzzle Game!

Data Thief is a 3D VR puzzle game set in a technology-themed world where the player must solve a series of laser grid puzzles in order to steal priceless data. The more high priority the hard drive, the harder it is to bypass security measures!

What I worked on:

This game was originally conceived as part of a final project in the Full Sail Game Development program. After graduating, myself and fellow classmates continued development and released the game for free on Steam! Published under our own OttercopterStudios LLC. I worked on the design and implementation of puzzles and interaction mechanics, as well as the majority of the UI. This project was built with Unity, using C#, for the HTC Vive.
