SIMETRI M3TS Care Under Fire Training

SIMETRI has developed a virtual environment surrounding a wounded soldier while engaged in active combat with a virtual enemy combatant. In this environment, the trainee must apply a tourniquet and return enemy fire when the situation calls for it. Augmented Reality (AR) plays a role by overlaying a pool of blood underneath the soldier’s wounded leg. The trainee’s weapon, medical instruments, and the wounded soldier provide haptic cues to the trainee while the virtual environment overlaid onto the physical scene and simulated physiology are provided by the M3TS system.

What I worked on:

I got to have a ton of fun working with the Microsoft Hololens 2 Augmented reality headset in order to deliver the initial phase one prototype for this project! I collaborated with several artists to create the AR blood particle system, and a few mechanical engineers to build the weapon controller and training tourniquet. This prototype was built in Unity using c#, and I wrote the embedded C++ code for all controllers and training devices. I developed my own Wifi and BLE networking code to allow all controllers: the weapon, tourniquet, and manikin, to work with each other!

SIMETRI CPR Training

SIMETRI created a module to assist health care professionals in the training and practice of CPR. This unit consisted of a box with a censor, connected to an app via Wifi. The application provides training instructors to see the compression depth and rate in real time.

What I worked on:

I was the lead developer on this project. I created the main application in Unity using C#, and the embedded software using C++. Not only did I write all the code for this project and implement all UI assets, I also soldered, flashed, and assembled all units! I immensely enjoyed traveling to local hospitals to deliver the finished project!
